Artist Quiz

A web application which quizes the user about a artist of their choosing via a multiple choice quiz

Technologies Used
- HTML5
- CSS#
- JavaScript
- Jquery
- Google fonts
- MusicBrainz api

Future Enhancements
- Better search functionality, the ability to see a dropdown menu
- More Questions
- More aesthetic and reactive front-end
